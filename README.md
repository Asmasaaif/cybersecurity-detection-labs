# Cybersecurity Detection Labs

This repository contains hands-on cybersecurity labs focused on SOC analysis, SIEM investigation, and threat detection.

## Projects

### 1. SIEM Alert Analysis (Suricata + Kibana)
- Analyzed 199+ SIEM alerts using Suricata and Kibana
- Investigated network activity and identified suspicious behavior (Mirai scanning)
- Correlated HTTP and flow logs to validate alerts
- Performed threat intelligence enrichment (AbuseIPDB, VirusTotal)
- Made incident response decisions (escalation vs non-escalation)

---

### 2. Windows Incident Analysis
- Investigated security events in a Windows environment
- Analyzed logs and identified suspicious system activity
- Performed initial incident triage and validation

---

### 3. Network Intrusion Detection
- Worked with intrusion detection concepts and traffic analysis
- Identified patterns of malicious network activity
- Analyzed alerts and network behavior

---

##  Tools & Technologies
